## 简介

一个可靠、美观、功能强大的WordPress的markdown插件，同时支持支持编辑和渲染，它是为了取代[WPEditor.md](https://wordpress.org/plugins/wp-editormd/)编辑器而开发的，成为下一代markdown编辑器。

它是基于块编辑器开发的，尽量利用块编辑器带来的强大扩展性，在此向wordpress的块编辑器开发者表示感谢。

已正式进入开发测试阶段。
