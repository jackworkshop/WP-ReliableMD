# WP-ReliableMD

## 简介

一个可靠、美观、功能强大的WordPress的markdown插件，同时支持支持编辑和渲染，为了替换一只猫的编辑器而开发，成为下一代markdown编辑器 

编辑器正在开发中，未正式发布，不建议使用

## 发布状态

**未发布**

## 特性

- [x] 基础Markdown
- [ ] 支持公式渲染
  - [x] Katex
  - [ ] MathJax
- [x] 前端+后端，二次渲染支持
- [ ] 可扩展 定制主题
- [ ] 支持UML
  - [ ] 支持mermaid
  - [x] 支持自带的UML绘制
- [ ] 插件兼容
  - [ ] WP Editor.md
  - [ ] WP HyperMD
- [x] 支持图标绘制
- [x] 表格扩展
- [x] 支持shortcode解析markdown
- [x] 支持前端渲染结果走后端缓存，替换后端预渲染结果缓存的功能
- [x] 支持缓存时间过期
